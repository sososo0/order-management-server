# 📖 프로젝트 소개

### ✨ 프로젝트 목적/상세
배달 및 포장 음식 주문 관리 플랫폼 개발을 위한 API 서버 개발 프로젝트.
광화문 근처에서 운영될 음식점들의 배달 및 포장 주문 관리, 결제, 그리고 주문 내역 관리 기능을 제공하는 플랫폼의 백엔드 API 개발
